## Projects

### Project 2 — User Management REST API

A RESTful API built using Node.js and Express that performs CRUD operations on users using in-memory storage.

**Features**

* Create User
* Read Users
* Update User
* Delete User
* Input Validation
* Error Handling

### Project 3 — User Management API with SQLite

An enhanced version of Project 2 that uses SQLite for persistent data storage.

**Features**

* SQLite Database Integration
* Full CRUD Operations
* Validation Middleware
* Persistent Storage

### Project 4 — Frontend & Backend Integration

A full-stack application that connects a frontend interface with the backend API developed in Project 3.

**Features**

* Display Users Dynamically
* Add New Users
* Edit Existing Users
* Delete Users
* Frontend Validation
* Backend API Integration
* Error Handling
